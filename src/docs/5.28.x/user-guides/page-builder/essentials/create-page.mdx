Webiny's page builder incorporates a drag-and-drop visual editor that allows you to easily create and edit web pages without requiring any coding skills.

In this tutorial, we will learn how to create a page in the page builder. As an example, we will create a page with the section shown in the images below:

*Image of Section that we will create in this tutorial*

We will do this in 7 steps:

- Step 1: Add and edit a new page
- Step 2: Add and edit a block
- Step 3: Add a heading
- Step 4: Add an image
- Step 5: Make the page responsive
- Step 6: Review settings
- Step 7: Preview and publish

## Step 1: Add a New Page

1. From the **Side Menu**, click **Page Builder** > **PAGES** > **Pages**.

2. Click **+ NEW PAGE**.

    ‚úîÔ∏è The **Select a category** screen opens.

3. In the **category** list, click **Static**.

    ‚úîÔ∏è Page editor screen opens.

4. Click the page's title **Untitled**.

    ‚úîÔ∏è A textbox appears.

5. In the textbox, update the text from **Untitled** to **The benefits of a Serverless CMS**.

6. Click anywhere outside the textbox.

    ‚úîÔ∏è The message "Page title updated successfully!" displays.


## Step 2: Add and Edit a New Block

1. Click the **ADD BLOCK** button ( ‚ûï ) at the bottom-right corner of the page editor.

    ‚úîÔ∏è The **All blocks** screen opens.

2. Hover on the **EMPTY BLOCK**.

3. Click **+ CLICK TO ADD**.

    ‚úîÔ∏è The page editor screen opens with the newly added block.

    **Note**: A block contains a grid, and a grid contains one or more cells.

4. Hover on and click the **grid** inside the empty block.

    ‚úîÔ∏è Several properties of the **grid** display in **STYLE** tab.

5. In the **STYLE** tab, click **Width**.

    ‚úîÔ∏è The **Width** accordion opens.

    a. In the **Width** value textbox, update the text from **1100** to **100**.

    b. In the **Width** unit dropdown, click **%**.

    **Note**: This will make the grid occupy 100% width of its parent element - the block.

6. In the **STYLE** tab, click **Grid Size**.

    ‚úîÔ∏è The **Grid Size** accordion opens.

    a. In the layout options, click the two column layout.

    b. Update the value of **Cell 1** from **6** to **5**.

    c. Update the value of **Cell 2** from **6** to **7**.

    **Note**: The cells are in order from the left to the right i.e. Cell 1, Cell 2, Cell 3, and so on.


## Step 3: Add a Heading

1. Click the **ADD ELEMENT** icon ( ‚ûï ) at the top-left corner of the page editor.

    ‚úîÔ∏è The elements menu opens.

2. Click **Basic**.

    ‚úîÔ∏è The **Basic** elements accordion opens.

3. Hover on **HEADING**.

4. Drag and drop the **HEADING** element inside **Cell 1**.

5. Double click the **Heading** to edit it.

6. Update the text from **Heading** to **The benefits of a Serverless CMS**.

7. Select the whole text by pressing **Ctrl + A** (or with the mouse).

8. In the text formatting menu that appears above the selected text, click the **bold** icon ( **B** ).

9. In the **STYLE** tab, click **Text**.

    ‚úîÔ∏è The **Text** accordion opens.

    a. In the **Color** menu, click the **Orange** color.

    b. In the **Heading Type** dropdown, click **H1**.

    c. In the **Typography** dropdown, click **Heading 3**.

10. In the **STYLE** tab, click **Margin**.

    ‚úîÔ∏è The **Margin** accordion opens.

    a. Click the **LINK ALL SIDES** icon ( üîó ).

    b. Update the margin-bottom value from **0px** to **60px**.

11. Now, following the Steps 1 to 10, add the following two texts with the mentioned properties below the current heading inside **Cell 1**:

    | Text                                                                                      | Formatting            | Color  | Heading Type | Typography | Margin |
    | :---------------------------------------------------------------------------------------- | :-------------------- | :----- | :----------- | :--------- | :------------ |
    | **Increase your business** <br/>**resiliency and reduce** <br/>**infrastructure spend** | Bold                  | Black  | H2           | Heading 2  | Bottom - 40px |
    | By switching to **Webiny** these are the <br/>**quantifiable gains** you can expect:| Normal + Bold (specified) | Orange | H3           | Heading 4  | Bottom - 60px |


## Step 4: Add an Image

1. From the **Basic** elements, drag and drop the **IMAGE** element inside **Cell 2**.

    ‚úîÔ∏è The media manager screen opens.

2. Click **UPLOAD**.

    ‚úîÔ∏è Your operating system's file explorer screen opens.

3. From the file explorer, upload this(*Add link to the image stored on a public directory*) image.

4. Click the uploaded image in the media manager.

    ‚úîÔ∏è The page editor screen opens.

5. In the **STYLE** tab of the image, click **Image**.

    ‚úîÔ∏è The **Image** accordion opens.

    a. In the **Width** value textbox, type **80**
    
    b. In the **Width** unit dropdown, click **%**.

6. In the **STYLE** tab of the image, click **Horizontal align**.

    ‚úîÔ∏è The **Horizontal align** accordion opens.

    a. Click the **ALIGN RIGHT** icon ( *Align right icon required* ).


## Step 5: Make the Page Responsive

1. Click the **Mobile portrait** icon.

    ‚úîÔ∏è Mobile view opens.

2. Click the **The benefits of a ...** text.

3. Click the **Text** accordion in the **STYLE** tab.

4. In **Alignment**, click the **CENTER** icon ( *Center Alignment icon required* ).

5. Similarly, align the **Increase your business ...** and **By switching to ...** text in center.

6. Following the Instruction 6 of Step 4, align the image in center.


## Step 6: Review settings

1. Click the **SETTINGS** icon ( ‚öôÔ∏è ).

    ‚úîÔ∏è The **Page Settings** screen opens.

2. In the **Title** textbox, type **Webiny | Home**.

3. In the **Path** textbox, type **/webiny-home**.

4. Click **SAVE SETTINGS**.

    ‚úîÔ∏è The page editor screen opens with the message "Settings saved!".
