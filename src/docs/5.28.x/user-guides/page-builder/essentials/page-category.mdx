In this tutorial, we will learn how to create a new page category. We will also learn how to assign a category to a page. 
As an example, we will create a page category named **blog** and assign it to a new page.

We will do this in 2 steps
- Step 1: Create a new page category
- Step 2: Create a new page under the category


## Step 1: Create a New Page Category

1. From the **Side Menu**, click **Page Builder** > **PAGES** > **Categories**.

2. Click **+ NEW CATEGORY**.

    ✔️ The **New Category** screen opens.

3. In the **Name** textbox, type **Blog**.

4. In the **Slug** textbox, type **slug**.

5. In the **URL** textbox, type **/blog/**.

6. In the **Layout** dropdown, click **Static page**.

7. Click **SAVE CATEGORY**.

    ✔️ The message "Category saved successfully." displays.


## Step 2: Create a New Page Under the Category

1. From the **Side Menu**, click **Page Builder** > **PAGES** > **Pages**.

2. Click **+ NEW PAGE**.

    ✔️ The **Select a category** screen opens.

3. In the **category** list, click **Blog**.

    ✔️ Page editor screen opens.

4. Click the page's title **Untitled**.

    ✔️ A textbox appears.

5. In the textbox, update the text from **Untitled** to **Blog Home**.

6. Click the **SETTINGS** icon ( ⚙️ ).

    ✔️ The **Page Settings** screen opens.

    **Note**: You can see in the **Path** textbox, **/blog/** is prepended to the path URL of the page.

7. In the **Path** textbox, update the text to **/blog/blog-home**.

8. Click **SAVE SETTINGS**.

    ✔️ The page editor screen opens with the message "Settings saved!".

9. Click **PUBLISH**.

    ✔️ The **Pages** screen opens with the message "Your page was published successfully!".

10. In the **PAGE PREVIEW** tab of the **Blog Home** page, click the **Kebab Menu**.

    ✔️ Menu opens with the options **View** and **Set as homepage**.

11. Click **View**.

    ✔️ The **Blog Home** page opens in a new tab.

    **Note**: You can see the URL in the address bar; the page's title is prepended by the term **/blog/**.
