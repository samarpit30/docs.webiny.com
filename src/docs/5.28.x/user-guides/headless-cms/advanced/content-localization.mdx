Webiny offers content localization that allows you to adapt your content to different languages by creating multiple locales.
For each locale, you can create content model groups, content models, content entries, etc., separately.

In this tutorial, we will learn how to create a content model for multiple locales.
We will do this in 2 steps:
- Step 1: Create and select a new locale.
- Step 2: Create a content model group and a content model in the new locale.

## Step 1: Create and Select a New Locale

We will create a new locale for the **Italic** language.

1. From the **Side Menu**, Click **Settings** > **Locales**.
2. Click **+ NEW LOCALE**.

   ‚úîÔ∏è The **New locale** section opens.

3. In the **Code** dropdown, type **it**.
4. From the available options in the dropdown, click **it-IT**.
5. Click **SAVE LOCALE**.
6. In the ** LOCALE** dropdown, click **it-IT**.

    **Note**: On creating and selecting a new locale, you will not find any content you created before. You need to add content schema and content to each locale separately.


## Step 2: Create a Content Model Group and a Content Model in the New Locale
If you are not familiar with how to create a content model group or a content model, please follow the [Create Content Model Group](../essentials/create-content-model-group) and [Create Content Model](../essentials/create-content-model) tutorials.
