Webiny offers content localization that allows you to adapt your content to different languages and locations by creating multiple locales.
The CMS provides you the flexibility to define a totally different content schema for different locales. That means for each locale, you can create content model groups, content models, content entries, etc., separately.

**Note**: On creating a new locale, you get a clean instance of the CMS for it. You need to add content schema and content to each locale separately. You will not find any previously created content in a new locale.

In this tutorial, we will learn how to create a content model for multiple locales.
We will do this in 4 steps:
- Step 1: Create and select a new locale.
- Step 2: Create a content model group in the new locale.
- Step 3: Clone a content model into the new locale.
- Step 4: Create a content entry in the new locale.


## Prerequisites

To follow this tutorial, you need the **Product** content model to clone it.
If you don't have the **Product** content model that we created in the [Create Content Model](../essentials/create-content-model) tutorial, please create it.


## Step 1: Create and Select a New Locale

We will create a new locale for the **Italic** language.

1. From the **Side Menu**, Click **Settings** > **Locales**.
2. Click **+ NEW LOCALE**.

## Step 3: Clone a Content Model into the New Locale

In this step, we will clone the **Product** content model from the **en-US** locale into the **it-IT** locale. We will update the clone content model's attributes with the following values:

| Attribute   | Values                                    |
| :-------    | :----------                               |
| Name        | **Prodotto**                              |
| Group       | **Commercio Elettronico**                 |
| Description | **Modello di contenuto del prodotto demo**|

**Note**: You can also [create a content model](../essentials/create-content-model) from scratch in the **it-IT** locale if you want.

1. In the ** LOCALE** dropdown, click **en-UN**.

2. From the **Side Menu**, Click **Headless CMS** > **Models**.

3. Hover on the **Product** content model.

4. Click the **CLONE CONTENT MODEL** icon ( *clone icon required* ).

    ‚úîÔ∏è The **Clone Content Model** screen opens.

5. In the **Name** textbox, update the text to **Prodotto**.

6. In the **Content model locale** dropdown, click **it-IT**.

7. In the **Content model group** dropdown, click **Commercio Elettronico**.

8. In the **Description** textbox, update the text to **Modello di contenuto del prodotto demo**.
    *Description textbox is uneditable in the CMS.*

9. Click **+ CLONE**.

10. Update the lables of the fields of the **Prodotto** content model with the following values:

    | Old Label    | New Label        |
    | :------------| :----------------|
    | Name         | **Nome**         |
    | Description  | **Descrizione**  |
    | Price        | **Prezzo**       |
