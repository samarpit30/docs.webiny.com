In this tutorial, we will learn how to update a [content model](../essentials/glossary#content-model). 
As an example, we will update the Product content model we created in the [Create Content Model](../essentials/create-content-model) tutorial.



## Update Content Model
1. From the **Side Menu**, Click **Headless CMS** > **Models**. 

   ✔️ The Content Models screen opens.

2. Hover on the **Product** content model.
3. Click the **EDIT CONTENT MODEL** icon ✏️.
   
   ✔️ The Product Content Model screen opens.

4. Click the Settings icon ⚙️.

   ✔️ The Content Model Settings screen opens.

5. To update the model's description, in the **General settings section**, click the **Content model description** textbox. 
6. Update its value to - "Demo Product Content Model for E-Commerce project".

   **Note**: Similarly you can update the name of the model.

7. To update the model's group, in the **General settings section**, click the **Content model group** dropdown.
8. If you have already created another group, select that group.

9. Click **SAVE SETTINGS**.
